For the main keyword SenSEO uses the first keyword in keywords meta tag.

The number of pages to be crawled can be configured in Add-ons Manager.

The exported values are seperated by the pipe "|" symbol.

Ranking is calculated with the following rules:

  • Positives
    • URL contains main keyword: +1
    • Title exists: +1
    • Title contains main keyword: +1
    • Description exists: +1
    • Description contains main keyword: +1
  • Negatives
    • Title has wrong length (20 < length > 65): -1 (tl)
    • Duplicate title: -1 (td)
    • Description has wrong length (50 < length > 150): -1 (dl)
    • Duplicate description: -1 (dd)

Press "Start crawling" button to analyze pages.
